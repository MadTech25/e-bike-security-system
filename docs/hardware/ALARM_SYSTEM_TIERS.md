# QED Alarm System — Product Tiers

## Basic
- Alarm
- Buzzer
- LED
- Tamper sensor
- Display

## Mid
- Alarm
- Tilt sensor
- Flashing LED
- Turn signals
- RFID *or* keypad

## Deluxe
- Alarm
- Tilt sensor
- Flashing LED
- Turn signals
- RFID + keypad

## Add-On
- Ultrasonic sensor for approaching vehicles

## Design Approach
- Single hybrid board for all tiers.
- Upgrades via added wiring or software.
- Case targets direct e-bike integration.
